# Contributing to Quick Menu

Thanks for your interest in Quick Menu! Here's how you can help.

## Reporting Bugs

Use the [Bug Report](https://github.com/Bassounet/QuickMenu-Plugin/issues/new?template=bug_report.yml) issue template. Please include:

- Your **Unreal Engine version** and **Quick Menu version**
- **Steps to reproduce** the bug
- Your **graph setup** (screenshot or description of which nodes/connections)
- Any relevant lines from the **Output Log** (filter by `LogQuickMenu`)

## Requesting Features

Use the [Feature Request](https://github.com/Bassounet/QuickMenu-Plugin/issues/new?template=feature_request.yml) issue template. Describe:

- **What** you'd like to see
- **Why** — what workflow would it improve?
- **How** you currently work around it (if applicable)

## Getting Help

- **Documentation:** [docs.quickmenuplugin.com](https://docs.quickmenuplugin.com)
- **Discord:** [Join the server](https://discord.gg/bThEnFzy) — ask in #help
- **AI-assisted help:** Paste [llms-full.txt](https://docs.quickmenuplugin.com/llms-full.txt) into ChatGPT/Claude along with your question

## Code Contributions

Quick Menu is a commercial plugin. The source code included with your purchase is for **personal modification and learning**. Pull requests are not accepted at this time, but your bug reports and feature ideas directly shape the roadmap.

## Code of Conduct

Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before participating.
