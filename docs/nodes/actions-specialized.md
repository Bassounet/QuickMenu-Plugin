# Action Nodes — Specialized Editors

## Material Op

Material editor operations.

**Options:** Apply Changes, Refresh, Delete Node, Clean Unused, Preview: Plane/Cylinder/Sphere/Cube/Preview Mesh, Toggle Live Preview, Toggle Stats.

## Sequencer Op

Sequencer playback and controls.

**Options:** Play Forward, Play Reverse, Stop, Go to Start, Go to End, Toggle Loop, Set Playback Speed (0.1x/0.25x/0.5x/1x/2x), Toggle Auto Key, Toggle Auto Scroll, Save, Render Movie, Toggle Snap.

## Static Mesh Op

Static mesh editor operations.

**Options:** Auto Collision, Simple Collision, Complex Collision, Remove Collision, Auto Generate LODs, Remove LODs, Add Socket, Rename Socket, Delete Socket, Enable Nanite, Disable Nanite, Refresh.

## Niagara Op

Niagara system controls.

**Options:** Compile, Apply, Reset, Toggle Loop, Speed (0.25x/0.5x/1x/2x), Save, Auto Compile, Toggle Bounds.

> Only visible when the Niagara plugin is loaded.

## PCG Op

PCG (Procedural Content Generation) graph operations.

**Options:** Execute, Clean, Flush Cache, Refresh, Toggle Debug.

> Only visible when the PCG plugin is loaded.
