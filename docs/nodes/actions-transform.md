# Action Nodes — Transform

## Transform Mode

Sets the transform gizmo mode.

**Options:** Translate, Rotate, Scale.

## Coord Space

Sets the coordinate system for transforms.

**Options:** World, Local, Parent.

## Select Op

Selection operations in the viewport.

**Options:** Select All, None, Invert, by Class, by Material, by Hierarchy, Children, Parent, Next, Previous.

## Reset Transform

Resets part or all of the selected actor's transform.

**Options:** Reset All, Location, Rotation, Scale, Snap to Grid.
